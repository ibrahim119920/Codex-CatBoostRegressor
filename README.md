# Prediksi Tinggi Muka Air dengan CatBoostRegressor

Model prediksi tinggi muka air berbasis `CatBoostRegressor`, menggunakan fitur lingkungan, kalender, dan jaringan sungai.

## Hasil

**RMSE: 1.61079**

## Isi repositori

- `notebook-150726.ipynb`: pipeline utama untuk audit data, validasi, training, dan pembuatan submission.
- `notebook-template.ipynb`: notebook template/baseline.
- `compare_submission.ipynb`: analisis perbandingan hasil submission.
- `architecture.md`: rancangan model dan pipeline secara lengkap.
- `logs.md`: catatan pengembangan dan verifikasi.
- `requirements.txt`: dependensi Python.

Dataset, hasil submission, model terlatih, dan artefak handoff tidak disertakan dalam repositori.
